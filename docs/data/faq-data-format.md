# FAQ 数据提交格式（给数据整理同学）

请按 `docs/data/faq-300-template.csv` 提交，字段固定如下：

1. `分类`：只能是 `社保` / `公积金` / `计划生育`
2. `标准问题`：必填，建议 8~40 字
3. `相似问法1~3`：至少填写 2 个；可留空 1 个
4. `标准答案`：必填，建议 20~220 字，使用官方口径
5. `状态`：`启用` 或 `停用`，默认 `启用`

数据质量要求：
- 不重复：同分类内标准问题不重复
- 不歧义：问题语义边界清晰
- 可回答：答案必须可直接返回给用户
- 可检索：相似问法覆盖口语表达

推荐配比：
- 社保：120
- 公积金：100
- 计划生育：80
